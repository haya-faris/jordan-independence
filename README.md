# Jordan Independence Day 🇯🇴

A React project celebrating Jordan's Independence Day.  
Includes history, gallery, quotes from kings, and messages from Jordanians.

## 📁 Pages:
- Home 🏠
- History 📜
- Gallery 🖼️
- Quotes 👑
- Messages ✉️

## ⚙️ Technologies Used:
- React + JSX
- MUI (Material UI)
- Node.js (Express backend)
- JSON for data
